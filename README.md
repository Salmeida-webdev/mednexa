# 🌍 MedNexa OET

Landing page premium internacional desenvolvida para um curso preparatório OET voltado para médicos, enfermeiros e profissionais da saúde que desejam validar seus diplomas e atuar internacionalmente.

O projeto foi construído com foco em:

- experiência premium
- design healthcare internacional
- alta conversão
- responsividade avançada
- microinterações modernas
- visual SaaS/EduTech

---

# 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Responsividade avançada
- Scroll animations
- Hover interactions
- Microinterações premium
